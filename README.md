# Vent Space

An anonymous venting web app where users can post thoughts and support others
without accounts or identities.

## Features
- Anonymous posting
- One support per user (anonymous auth)
- Support toggle
- Firestore backend
- Secure rules to prevent abuse

## Tech Stack
- Next.js
- Firebase (Auth + Firestore)
- shadcn/ui
- Tailwind CSS

## Status
UI polish in progress. Core functionality complete.
